## About Us
Rewatch Plus is a social network project initiative focused on unifying cinematic experiences between people, combining profiles to watch films together, express your opinions on the cinema and customize your profile however you want.

# rewatch-api
API da aplicação
## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
